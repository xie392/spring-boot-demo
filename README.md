## 项目简介

本项目是一个基于 `Spring Boot` 后端 + `Vue3` 前台的项目，主要用于学习 Spring Boot 的使用。后端简单使用了 `Mybatis Plus`，前台使用了 `Vue3` + `Element Plus`。
